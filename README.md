# Human Activity Clustering with K-Means

## 📌 Descripción del proyecto

Este proyecto aborda un problema de aprendizaje no supervisado aplicando el algoritmo K-Means sobre datos de actividad humana. El objetivo es identificar patrones latentes en los datos y explorar cómo diferentes observaciones pueden agruparse en función de su comportamiento.

## 🎯 Motivación

Este trabajo forma parte de mi proceso de formación en ciencia de datos, con un enfoque particular en el aprendizaje de técnicas de Machine Learning. Me interesa comprender no solo la implementación de modelos, sino también su interpretación y utilidad en contextos reales.

## 📊 Dataset

Se utiliza el dataset de Human Activity Recognition (HAR) disponible en el repositorio de UCI.

Debido a restricciones de tamaño, el dataset no se incluye en este repositorio.
Puede descargarse desde la fuente oficial:
https://archive.ics.uci.edu/ml/datasets/human+activity+recognition+using+smartphones

## ⚙️ Metodología

* Preprocesamiento y limpieza de datos
* Escalamiento de variables
* Aplicación del algoritmo K-Means
* Evaluación de la calidad de los clusters
* Análisis e interpretación de resultados

## 📈 Resultados y enfoque analítico

El modelo permite identificar agrupaciones consistentes dentro de los datos, lo que facilita la comprensión de estructuras subyacentes en las actividades humanas.

Más allá del resultado técnico, el foco está en la **accionabilidad de los hallazgos**, es decir, en cómo los patrones detectados pueden traducirse en información útil para la toma de decisiones o el entendimiento del comportamiento.

## 🛠️ Tecnologías utilizadas

* RPubs
* Algoritmo K-Means
